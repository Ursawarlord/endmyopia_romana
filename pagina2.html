  1. Diopterele sunt contoare inverse (amintiți-vă 100cm = 1m) D = 1 / Metri; Invers metre = 1 / D
  2. Punctul de refracție și * ghid pentru rezistența diferențială * (depinde în continuare de distanță / unele persoane fac excepții) https://youtu.be/9z2jjT_Gm7o
    s = distanța față de obiect (pentru distanță doar pusă în număr mare) (metri)
    s '= distanța până la punctul de refracție (pentru miopi cât de departe se poate vedea clar) (metri) (introdus ca negativ)
    f = distanța focală a obiectivului (metri)
    P = puterea obiectivului (dioptrii)
    1 / f = (1 / s) + (1 / s ')
    1 / f = P
  3. Ecuația vizuală a acuității (unități consistente) (Înălțimea fontului / distanța de semnat) (180 / 60pi) = arcminute = a
    Notă: 5Arcminute = 20/20
    Setați proporția a / (20 / x) = 5 / (20/20)
  4. Acuitatea vizuală (mm / metri) (Înălțimea fontului {mm} / distanța de semnare {m}) * 13.75 = numitorul x de 20 / x
  5. Acuitatea vizuală (în / picioare) (Înălțimea fontului {în} / distanța de semnat {ft.}) * 1146 = numitor x de 20 / x
